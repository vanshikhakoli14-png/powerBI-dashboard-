# 📊 Power BI Sales Dashboard

## 📌 Project Overview
This project presents a Sales Dashboard built using Power BI to track business performance, compare yearly sales, and analyze growth trends.

## 🚀 Key Features
- 💰 Current Year Sales: 723.71M
- 📊 Last Year Sales: 566.13M
- 📈 YoY Sales Growth: 27.84%
- 🎯 Budget Analysis with variance tracking
 
- ## 🛠 Tools & Technologies
- Power BI
- Data Modeling
- DAX (for calculations)
- Excel (data source)
